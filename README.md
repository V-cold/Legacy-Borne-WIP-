# Legacy Borne 0.4.0 Pre-alpha
This repository is open source and made to be modular
use this however you wish, but I do not consent to this
being used to train AI. I hope you can respect that.

### To create an environment suitable to build your 3DS Deltarune Rom, you will need,
1. The UndertaleModCli (https://github.com/UnderminersTeam/UndertaleModTool)
2. 3DSkit (https://github.com/Tyulis/3DSkit)
3. A copy of Deltarune (https://store.steampowered.com/app/1671210/DELTARUNE/)
4. A Nintendo 3DS or Nintendo New 3DS (or equivalent emulator)
5. Linux (If any one wants to fork this and make it work on windows go for it)

### Setting up your environment
1. Clone this repository into an empty folder.
2. Move the file named UndermodCLI into this folder.
3. Clone 3DSkit into the same repository
4. Have steam install DELTARUNE into this folder.
5. Run roots.py via the command "python 3 roots.py"
